# Veteran Benefits Assistant — Claude Chatbot

A browser-based chatbot that answers questions about VA benefits, powered by the Claude API.

## What it does
- Answers questions about disability claims, GI Bill, VR&E, TDIU, VA healthcare, home loans, and appeals
- Maintains conversation history for multi-turn dialogue
- Includes quick-start suggestion buttons for common questions

## How to run

1. Get an Anthropic API key at https://console.anthropic.com
2. Open `index.html` in a browser
3. Enter your API key in the field at the top
4. Start asking questions

No server needed — runs entirely in the browser.

## Example Questions
- "How do I file a VA disability claim?"
- "What is VR&E Chapter 31 and do I qualify?"
- "How does Post-9/11 GI Bill housing allowance work?"
- "What is TDIU and how do I apply?"

## Tech
- HTML / CSS / JavaScript
- Anthropic Claude API (claude-sonnet-4-6)
- Conversation history management
