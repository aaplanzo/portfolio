# Antonio Planzo — Portfolio Website

A personal portfolio site built from scratch with HTML and CSS. No frameworks, no templates.

## What it includes
- Hero section with identity and tags
- About section with background summary
- Project grid linking to all 6 portfolio projects
- Contact section

## How to run
Open `index.html` in any browser. No server needed.

## Notes
- Update the email address in the contact section
- Project cards for #4–6 will activate as those projects are completed
- Designed to live at `03_website/` inside the portfolio repo
